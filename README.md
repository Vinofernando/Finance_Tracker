# 💰 Finance Tracker App

A full-stack personal finance management application built to help users track their income and expenses in real-time.

## 🚀 Live Demo
- **Frontend:** [https://finance-tracker.store](https://finance-tracker.store)
- **Subdomain Demo:** [https://portofolio.finance-tracker.store](https://portofolio.finance-tracker.store)

## 🛠️ Tech Stack

### Frontend
- **React.js** (Vite)
- **Lucide React** (Icons)

### Backend
- **Node.js**
- **Express.js**
- **PostgreSQL** (Database)
- **JSON Web Token (JWT)** (Authentication)
- **Bcrypt** (Password Hashing)

### Infrastructure
- **Netlify** (Frontend Hosting)
- **Supabase** (Database)
- **Railway** (Backend & Database Hosting)
- **Cloudflare** (DNS Management & SSL/TLS)

## ✨ Key Features
- **User Authentication:** Secure login and register system.
- **Transaction Management:** Create, Read, Update, and Delete (CRUD) for income and expenses.
- **Dynamic Dashboard:** Real-time summary of total balance, income, and spending.
- **Responsive Design:** Mobile-friendly interface.

## ⚙️ Local Setup

1. **Clone the repository**
   ```bash
   git clone [https://github.com/Vinofernando/Finance_Tracker.git](https://github.com/Vinofernando/Finance_Tracker.git)

2. Install Dependencies
  # For Frontend
  cd Frontend
  npm install

  # For Backend
  cd ../Backend
  npm install

3. Configure Environment Variables
   PORT=3000
  DB_URL=your_postgresql_url
  JWT_SECRET=your_secret_key

4. Run The Project
   # Backend
  npm start
  
  # Frontend
  npm run dev
